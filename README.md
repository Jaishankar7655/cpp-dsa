# C++ DSA

This project organizes basic C++ practice programs inside the `C++ DSA` workspace.
Each folder contains examples for core C++ topics such as arrays, loops, functions, and number systems.

## Folder Structure

- `array/` - array operations and search/min/max programs
- `Binary/` - binary and decimal conversion examples
- `bsicsloops.cpp/` - loops, pattern printing, and basic control flow
- `functions/` - examples of using functions in C++
- `vectors/` - examples using the C++ `vector` container

## Usage

- `*.cpp` files are source code.
- `*.exe` files are compiled binaries; use them only to run programs, not to edit code.
- Each folder includes a `README.md` that summarizes the files and explains the theory.
