# C++ DSA

यह प्रोजेक्ट `C++ DSA` के भीतर बेसिक C++ अभ्यास प्रोग्राम्स को व्यवस्थित करता है।
यहाँ हर फ़ोल्डर में अलग-अलग विषय और छोटे-छोटे उदाहरण हैं जो C++ के बुनियादी डेटा स्ट्रक्चर, लूप, फ़ंक्शन, और नंबर सिस्टम को समझने में मदद करते हैं।

## फ़ोल्डर संरचना

- `array/` - एरे ऑपरेशन्स और सर्च/मैक्स/मिनिमम के प्रोग्राम
- `Binary/` - बाइनरी और डेसिमल कन्वर्शन
- `bsicsloops.cpp/` - लूप, पैटर्न प्रिंटिंग और बेसिक कंट्रोल फ्लो
- `functions/` - फ़ंक्शन उपयोग के उदाहरण
- `vectors/` - C++ `vector` का उपयोग

## उपयोग

- `*.cpp` फाइलें स्रोत कोड हैं।
- `*.exe` फाइलें कंपाइल्ड बायनरी हो सकती हैं; इन्हें केवल रन करने के लिए उपयोग करें, कोड संपादन के लिए नहीं।
- प्रत्येक फोल्डर में `README.md` उस फ़ोल्डर की फाइलों का सारांश और थ्योरी देता है।
